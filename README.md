# Note Taker

## Description

The project is to create an application that can be used to write and save notes so that it can help users to track tasks need to complete. This application will use an Express.js back end server and will save and retrieve note data from a JSON file.

## Installation <a id="Installation"></a>
1. Install package.json with command `npm i`
2. Install express with command `npm install express --save`
3. Install uniqid with command `npm i uniqid`


## Usage <a id="Usage"></a>
1. Opening an external ternimal from the repository
2. Run the backend server locally with command: `npm start`
3. Access the application by `http://localhost:3001` in a web broswer on the local server
4. Deployed application on Heroku: https://note-taker-zl.herokuapp.com/
